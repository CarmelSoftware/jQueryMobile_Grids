Grids

This is a small App to exemplify Using Grids on jQueryMobile.

jQuery Mobile has a set of built-in CSS3 column layouts, meant to position buttons , navigation tabs, etc , as if it was in a table.

This is accomplished by using the ui-grid-solo|a|b|c|d classes for the parent div, and ui-block-a|b|c|d|e for the elements
